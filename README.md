## Project Type
Hands-on / Lab Project (Fresher Level)

## Domain
Insurance (BFSI)

## Objective
To simulate an insurance policy management application hosted on the cloud
and practice L1 IT Support activities such as monitoring, troubleshooting,
incident handling, and escalation.

## Project Description
This project represents a cloud-hosted insurance policy management system
used to manage policy creation, renewals, and claims processing.

The environment is simulated for learning purposes only.
No real customer or insurance data is used.

## Technologies Used
- AWS EC2, IAM, VPC, CloudWatch
- Linux
- API basics
- Incident Management (ITIL approach)

## L1 Support Activities
- Monitoring application health and server metrics
- Handling application accessibility issues
- Checking API availability
- Linux-level troubleshooting
- Incident logging and escalation

## Key Learnings
- Understanding insurance domain applications
- Role of IT Support in insurance systems
- Incident lifecycle management
- Importance of monitoring and SLA
