## Project Type
Hands-on / Lab / Practice Project (Fresher Level)

## Domain
Insurance (BFSI)

## Objective
To understand how an insurance policy management application is hosted,
monitored, and supported in a cloud environment from an L1 IT Support perspective.

## Project Description
This project simulates a cloud-based insurance policy management system
used for policy creation, renewals, and claim processing.

The environment is created only for learning purposes.
No real insurance company data or customer data is used.

## Technologies Used
- AWS EC2, IAM, VPC, CloudWatch
- Linux
- API basics
- Incident Management (ITIL concepts)

## L1 Support Activities Practiced
- Monitoring server and application health
- Basic Linux troubleshooting
- Handling application access issues
- API availability checks
- Incident logging and escalation

## Key Learnings
- Insurance domain application basics
- Role of IT Support in insurance systems
- Monitoring and alert handling
- Incident lifecycle management
